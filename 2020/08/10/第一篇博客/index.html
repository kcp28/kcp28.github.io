第一篇博客

#用hexo博客框架搭建静态博客#
1.下载安装node.js
因为hexo时基于node.js 的
查看版本：node -v
npm -v
2.借助包管理器来安装hexo
1>安装cnpm : npm的淘宝镜像源 用这个有点快。
查看版本：cnpm -v
2>安装hexo
cnpm install -g hexo-cli //-g 表示全局安装
查看版本：hexo -v
能查看到版本，表示安装成功。至此，hexo框架已经安装完成。
3.开始使用hexo，正式开始搭建博客（至此以后的操作可以在git Bash中进行）
1>建立博客目录blog：mkdir 路径/blog
2>进入blog目录下，使用hexo生成博客：hexo init

4.新建一篇博客
hexo s 启动预览
hexo n “博客名” 生成一篇新博客
